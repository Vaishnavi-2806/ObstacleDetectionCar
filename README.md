# Obstacle Detection Car Using Arduino and Ultrasonic Sensor

## Overview
The Obstacle Detection Car is an autonomous robotic vehicle designed using Arduino and an ultrasonic sensor. The system continuously measures the distance between the vehicle and nearby objects. When an obstacle is detected within a predefined range, the car automatically stops and changes its direction to avoid collisions.

## Features
- Real-time obstacle detection using an ultrasonic sensor.
- Automatic collision avoidance.
- Autonomous navigation without manual control.
- Cost-effective and easy-to-build design.
- Suitable for learning embedded systems and robotics.

## Working Principle

The Obstacle Detection Car uses an Arduino Uno and an HC-SR04 ultrasonic sensor to detect nearby obstacles. The ultrasonic sensor continuously measures the distance between the car and objects in its path. When an obstacle is detected within a predefined range, the Arduino sends signals to the motor driver and relay to stop or control the motors, preventing a collision. The vehicle is powered by a battery supply, while the switch is used to turn the system ON or OFF. This setup enables automatic obstacle detection and safe navigation.
